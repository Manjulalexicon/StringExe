# String Exercises in Java

This repository contains solutions to a series of Java string manipulation exercises. The code demonstrates common string operations such as length calculation, character access, substring extraction, case conversion, trimming, parsing, splitting, and converting between strings and character arrays.



## 📝 Exercises

1. **Length of a String** – Calculate the length of `"Java"`.
2. **Character at Index** – Find the character at index 6 in `"Long example sentence"`.
3. **Index of 'o'** – Find the first index of `'o'` in `"Even longer example sentence"`.
4. **Substring** – Extract `"not as long"` from `"Ok this is not as long!"`.
5. **Lowercase and Uppercase** – Convert `"CAPS EQUALS SCREAMING"` to lowercase and then back to uppercase.
6. **Trim String** – Trim `"\tJ\ta\tv\ta\t"`.
7. **Parse to String** – Convert `int 20` to a String and append `"20"` to produce `"2020"`.
8. **Split String (Oil and Water)** – Split `"Oil and Water"` into `"Oil"` and `"Water"`.
9. **Split String (Names)** – Split `"Carl,Susie,Fredrick,Bob,Erik"` into individual names.
10. **Convert to Char Array** – Convert `"ThisShouldBeConverted"` to a char array and iterate through it.
11. **Convert Char Array to String** – Convert `{'J','a','v','a'}` to a String.
